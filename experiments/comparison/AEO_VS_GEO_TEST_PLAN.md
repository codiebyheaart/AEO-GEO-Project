# AEO vs GEO Test Plan

## Objective

Compare which optimization style performs better for B2B SaaS paper print content across search-style and AI-style discovery.

## Assets Under Test

### AEO Assets
- FAQ definition page
- pricing page

### GEO Assets
- ROI guide
- security and compliance page

## Query Set

1. what is print procurement software
2. how much does print ordering software cost
3. roi of switching from manual print buying to print procurement software
4. is print procurement software secure for enterprise procurement teams
5. best print management software for enterprise teams
6. how to automate print ordering for distributed offices

## Channels

- Google Search
- ChatGPT
- Gemini
- Perplexity
- Copilot

## Scorecard

For each test, score from 1 to 5:
- visibility
- accuracy
- completeness
- citation quality
- brand mention
- conversion intent support

## Expected Pattern

- AEO should win direct answer and pricing queries
- GEO should win ROI, security, and comparison prompts
- combined content should win workflow and buyer-evaluation prompts

## Execution Steps

1. Use the exact same query wording each round.
2. Test all channels on the same day if possible.
3. Capture screenshots and outputs.
4. Log results in the experiment tracker.
5. Summarize results weekly.
