# 30-Day AEO + GEO Roadmap

This roadmap is designed to make you client-ready for AEO and GEO work using your live showcase project.

## Week 1: AEO Foundations

### Day 1

- Read your live AFTER page and identify every answer block.
- Write 10 question-led headings for one new niche.
- Exercise: convert 3 generic headings into direct buyer questions.

### Day 2

- Study featured snippets and People Also Ask results for 10 target queries.
- Exercise: write 5 snippet-ready answers between 40 and 60 words.

### Day 3

- Learn FAQ schema structure.
- Exercise: create 5 FAQs for a SaaS or service business.

### Day 4

- Learn HowTo schema structure.
- Exercise: turn one workflow into 5 ordered steps.

### Day 5

- Compare your BEFORE and AFTER pages.
- Exercise: list 10 AEO improvements visible on the AFTER page.

### Day 6

- Use Google to study search intent patterns.
- Exercise: classify 20 queries into definition, pricing, comparison, ROI, and how-to.

### Day 7

- Build one mini AEO page draft for a new topic.
- Exercise: write H1, 3 question-led H2s, one FAQ block, and one CTA.

## Week 2: GEO Foundations

### Day 8

- Learn what makes content citation-friendly in AI engines.
- Exercise: rewrite one paragraph using clearer entities, numbers, and decision context.

### Day 9

- Study how AI engines summarize comparison and ROI queries.
- Exercise: create one 2-column comparison table and one ROI table.

### Day 10

- Learn entity-based writing.
- Exercise: define product, audience, industry, geography, and use case for one brand.

### Day 11

- Learn how AI engines prefer structured definitions.
- Exercise: write 3 entity-rich definitions for one product.

### Day 12

- Practice trust and security content.
- Exercise: create one security section covering SSO, permissions, logs, and encryption.

### Day 13

- Practice AI prompt testing.
- Exercise: run 5 prompts in ChatGPT and 5 in Gemini using your page topic.

### Day 14

- Review which phrases or facts get reused by AI tools.
- Exercise: record 5 observations in the live verification scorecard.

## Week 3: Data and SEO Support

### Day 15

- Open Search Console and inspect your live page performance.
- Exercise: note impressions, clicks, CTR, and position for top queries.

### Day 16

- Review GA4 landing page behavior.
- Exercise: capture engagement rate, sessions, and CTA behavior for the page.

### Day 17

- Study title tags and meta descriptions of top-ranking competitors.
- Exercise: write 3 better title variations and 3 better meta descriptions.

### Day 18

- Validate schema markup using Google's Rich Results Test.
- Exercise: test FAQ and HowTo markup and note issues.

### Day 19

- Review heading hierarchy and internal link structure.
- Exercise: redesign one page outline for clarity and intent match.

### Day 20

- Check competitor content using Ahrefs or manual SERP review.
- Exercise: build a gap list of missing topics, missing facts, and missing questions.

### Day 21

- Create one combined AEO + GEO page for a new niche.
- Exercise: include answer blocks, tables, FAQ, and one proof section.

## Week 4: Client Readiness

### Day 22

- Practice using the client audit template.
- Exercise: audit a real website in your target niche.

### Day 23

- Prepare a 5-minute explanation of AEO vs GEO.
- Exercise: explain it in simple business language.

### Day 24

- Build a delivery checklist.
- Exercise: define what you deliver for Starter, Professional, and Premium packages.

### Day 25

- Practice prompt testing workflow.
- Exercise: run the exact same prompts across ChatGPT, Gemini, Perplexity, and Copilot.

### Day 26

- Build a before-and-after score summary.
- Exercise: compare one weak page with one optimized page using your scorecard.

### Day 27

- Prepare objection handling.
- Exercise: answer these questions:
  - How do you prove GEO?
  - How long does AEO take?
  - What if AI does not cite the brand yet?

### Day 28

- Practice turning findings into recommendations.
- Exercise: write a 7-day, 30-day, and 90-day action plan for one mock client.

### Day 29

- Build your client portfolio proof.
- Exercise: present the case study page, scorecard, and audit template together.

### Day 30

- Simulate a real client call.
- Exercise: present your offer, show proof, explain next actions, and close with a package recommendation.

## Output By Day 30

By the end of this roadmap you should have:

1. one working showcase project
2. one repeatable audit template
3. one live verification scorecard
4. one polished case study
5. one clear AEO + GEO service workflow

## Best Practice

Every 7 days, repeat the same prompts, compare results, and update your proof. Repetition matters more than reading theory.
